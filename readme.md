# AI Chatbot for Social Platforms 🤖

This project is an **AI-powered chatbot for social platforms** built using **Python and FastAPI**.  
It uses a **local open-source language model (TinyLLaMA via Ollama)** to generate intelligent and fast responses without relying on paid APIs.

---

## 📌 Project Objective

To design and develop an AI chatbot that can:
- Understand natural language
- Respond intelligently in real time
- Work across social platforms
- Operate without paid AI services
- Provide safe and reliable interactions

---

## 🛠 Technologies Used

- **Python 3.10**
- **FastAPI** – backend REST API
- **TinyLLaMA (via Ollama) + groq API + Serp API** – local AI language model
- **Telegram Bot API**
- **Uvicorn** – ASGI server
- **dotenv** – environment variable handling

---

## ✨ Features

- 🧠 Natural Language Understanding
- ⚡ Fast AI responses (lightweight model)
- 🌐 Multi-platform support (API + Telegram)
- 💬 Context-aware replies
- 🛡 Basic safety handling
- 🔐 No paid API or internet dependency for AI
